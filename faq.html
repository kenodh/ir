How to get a username and a password to login to student information system 

Student information system is only opened for the parents of the students who are currently in Royal College. It will not be opened to the public because it contains confidential information about the student. If you are a parent who want to get the username and a password please contact the class teacher of your son. When entering the username it sould be entered in the format of YYYY/XXXX. Part before the stroke is the year and it should be entered in four digits. Part after the stroke is a sequence number which should also be entered in four digits.

I get a message saying that I don't have the privileges to view the pages 

Still the cause of this problem is unknown and it occurrs when viewing this web site using slower connection. If you get such message try adding this page to the trusted sites of your web browser. Refer you web browser's help to see how to add a web site to the trusted sites of the web browser.

What are the best browsers to view this web site 

This web site is tested using Microsoft Internet Explorer 6, Mozilla Firefox 1.5 and Netscape Navigator 7.2 web browsers and later versions of them. It is recommended to use the above browsers but later versions of other browsers will be fine.

What should I do when there is wrong personal information about my son 

Student's personal information is collected using the "Student Record Sheet" which is available in the downloads section of this web site or could be acquired from the class theacher of your son. Please fill it with new details and submit it to the class teacher of your son. New information will be added to the system as soon as possible after the submission. We are not allowing the parents to update the son's information through web site for security reasons.

When I click Student Information link popup blocker reports that a popup from this site is blocked 

When Student Information link is selected, popup blockers in some browsers reports that a popup from the current web site has been blocked. Please see the browser documentation for allowing popups from specific web site. In Microsoft Internet Explorer it shows a information bar on top of the window. When you right click on that information bar it will give options to stop blocking popups from current web site.

My browser does not display the web page correctly 

This may happen because of two reasons. Your browser may be incompetible to the standards required to view this web page or your browser may be keeping an old copy of this web site. As the first solution you may try clearing the browser cache. See your browser's documentation for clearing the browser cache. If the problem persists try upgrading your browser to newer version.

I could not view videos correctly 

To watch the videos online you sould have the latest adobe flash player. It could be downloaded from the adobe web site. Viewing videos online also requires a broadband connection such as ADSL. If you can not view the video online correctly you could download the video using the download link provided. It will take you to the page where you would be allowed to download the video.
